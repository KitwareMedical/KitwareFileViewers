Open in… extension

How it works

File information (i.e. the filename and the file type/extension) is extracted from the URL by first parsing the URL and if it fails, by doing HTTP requests to guess the type of the file. With the collected information, a URL is generated for the chosen viewer to pass the URL to it.

3D Slicer

If you want to open your URLs with 3D Slicer, you need to install Slicer and save the LoadRemoteFile module into the qt-scripted-modules folder of 3D Slicer. It will work automatically on Windows.

There is more setup required on other OS.
